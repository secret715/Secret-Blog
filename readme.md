[Secret Blog](http://blog.gdsecret.net/)
=====
簡單、輕便的PHP+MySQL部落格系統



安裝
-----
1. 上傳文件後，執行http://你的網址/install.php
2. 填入相關資料
3. 安裝成功！



使用
-----
* 後台位置：http://你的網址/admin/index.php
* 詳細資料(如：網站名稱)：config.php
* 側邊欄：include/sidebar.php
